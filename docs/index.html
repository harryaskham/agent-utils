Publishing and preview

GitHub Pages path: Repository Settings → Pages → Deploy from branch: main, folder: /docs

Local preview: npm run docs:build && python3 -m http.server --directory docs 8000

Validation: npm run docs:check

The rendered page is static HTML/CSS generated from docs/tools.json, so GitHub Pages can publish it without secrets, build services, or local-only daemon state.
